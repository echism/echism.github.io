For a work sample I've chosen to draw from my graduate work for the following reasons:

  • It is properly mine. Sharing it will not violate anyone's privacy, IP or anything else along those lines.
  • I planned and executed it alone. I need not define complex roles or task delegation among a team to explain it.
  • It's interesting.
  • It's (relatively) recent

For a rather open ended cloud computing project, I elected to teach myself how to use autodoc vina with AWS.

I liked this particular topic because it allowed me to work with COVID-19 and explore the science around it in my own way.

Autodoc vina allows you to simulate molecular docking.

Many molecular models are available for download from the Protein Data Bank. This is where I obtained the model for COVID-19.

There are two major parts to this process. The first is preparing the receptors and ligands for processing. The second is running the cloud analysis
